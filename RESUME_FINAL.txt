================================================================================
                         INSTRUCTIONS DE DÉPLOIEMENT
================================================================================

OPTION 1: VIA GITHUB (RECOMMANDÉE)
1. Créer un repository GitHub
2. Pousser le code:
   git init
   git add .
   git commit -m "Initial commit: ExportiaSN website v1.0"
   git remote add origin https://github.com/yourusername/exportia-sn.git
   git push -u origin main

3. Sur Netlify:
   - Aller sur https://netlify.com
   - Cliquer "New site from Git"
   - Sélectionner le repository
   - Les paramètres seront auto-détectés

OPTION 2: DÉPLOIEMENT DIRECT
   npm install -g netlify-cli
   netlify login
   netlify deploy --prod
